# 結論

　本研究では，南鳥島周辺海域産マンガンノジュールに対し，硫酸浸出法，シュウ酸浸出法，アンモニア-炭酸アンモニウム溶液浸出法の3手法を比較し，回収性能と経済性を評価した．試料の主要成分はMn 16.11 wt %，Fe 17.00 wt %，有用金属はCo 3,712 ppm，Ni 2,008 ppm，Cu 1,654 ppmであった．

　硫酸浸出法では，未焙焼試料を用いた場合，Ni・Cuをほぼ100 %回収する一方，Coは約23 %にとどまり，Co回収が主要課題となった．しかし，600 ℃，空気雰囲気，炭素付加率10 wt. %で3時間の還元焙焼処理を行うことで，Mn^4+^をMn^2+^に還元し，結晶構造を変化させることで浸出性が大幅に改善された．焙焼試料を用いた硫酸浸出（75 ℃，2 M，180分）では，Niは94.8 %，Cuは98.3 %，Coは94.8 %の高い浸出率を示し，Co回収率の課題が解決された．さらに，Mnも61.4 %の浸出率で回収可能となった．シュウ酸浸出法（0.3 M，25 ℃，120分）はNi・Co・Cuをほぼ100 %回収したが，Mn・Feも溶出し，後処理でシュウ酸塩が生成してNi/Co/Cuも共沈してしまう可能性があり，技術課題が残ることが判明した．アンモニア-炭酸アンモニウム溶液浸出法（未焙焼，50 ℃，15 % NH~3~-9 % CO~2~，120分）はCu約46 %，Ni約7.5 %，Co約2.1 %と回収率が著しく低く，評価対象から除外した．

　経済性の比較では，試薬コスト，エネルギーコスト，処理時間，および有用金属の回収率に基づく収益性を総合的に評価した．硫酸浸出法（焙焼試料，75 ℃，2 M，180分）では，Ni，Co，Cuの回収価値は約195 USDとなる．これに加え，マンガンを硫酸マンガンとして回収することで追加で約247 USDの価値が得られ，合計回収価値は約442 USDとなった．これに対し，浸出用試薬157 USD，分離工程の主要試薬約67 USD，加熱・焙焼164～170 USDで総コスト約388～394 USD，差引約48～54 USD/tとなり，唯一収益性がプラスとなった．シュウ酸浸出法（0.3 M，25 ℃）は回収価値約222 USDに対し，試薬1,870 USD（加熱0 USD）で差引▲1,648 USD/tとなった．

　以上より，南鳥島のマンガンノジュールに対しては，焙焼→硫酸浸出→pH調整によるFe除去→段階的な分離・回収という処理フローが適していると考えられる．具体的には，まず還元焙焼処理により浸出性を向上させ，次に硫酸浸出によりNi，Co，Cuをほとんど全て浸出させる．その後の分離・精製については，硫酸系浸出液を対象に，pH調整によるFe除去の後，溶媒抽出でCu/Co/Niを段階的に回収し，最後に高pHでMnを沈殿回収するフローが提案されている[8]．本研究の浸出液に対しても同様の段階分離を適用することで，有用金属の回収と精製の両立が可能と考えられる．この処理フローにより，現段階で商業利用に向けた有望な選択肢となった．今後の検討課題としては，浸出しなかったMnの回収による収益性のさらなる改善，スケールアップ試験，環境対策コストの精査，および実用化に向けたプロセス最適化が挙げられる．
