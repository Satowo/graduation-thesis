# 研究手法

## 試料

　実験試料は，2024年に日本財団と東京大学の共同調査によって南鳥島海域で採取されたマンガンノジュール試料である[3]．本航海で得られたマンガンノジュール試料15個をジョークラッシャーで粉砕し，試験用ふるいで粒径を125 μm以下に揃えたものを化学分析および湿式処理試験に使用した．

## 試料の酸分解

　試料の元素組成を調べるため，過塩素酸，フッ酸，硝酸を用いた酸分解を行い，ICP-MSを用いて分析した．4つの分析結果の平均値を後述の実験での浸出率計算に用いた．

## 焙焼処理およびXRDによる構造分析

　マンガンノジュールの還元焙焼に伴う結晶構造の変化を評価するため，未焙焼試料および種々の条件下で焙焼処理を施した試料に対してX線回折（XRD）分析を行った．焙焼処理では，粉末試料に還元剤として黒鉛を添加し，マッフル炉を用いて加熱を行った．実験条件として，焙焼温度は500 ℃，600 ℃，700 ℃，800 ℃，雰囲気は大気（空気）および窒素，黒鉛の添加量（炭素付加率）は3，5，10 wt. %，保持時間は30，60，120，180分とした．

## 硫酸による浸出実験

　硫酸による浸出実験では，硫酸の濃度を0.5 M，1 M，2 Mの3種類，浸出温度を25 ℃，75 ℃，100 ℃とした．未焙焼試料および焙焼試料（600 ℃，空気雰囲気，炭素付加率10 wt. %，3時間）を用いて実験を行った．浸出開始から15分，30分，60分，120分，180分経過した時点で溶液を採取し，ICP-MSで各元素濃度を分析した．

## シュウ酸による浸出実験

　シュウ酸による浸出実験では，未焙焼のマンガンノジュール試料を用い，シュウ酸濃度を0.1 M，0.3 M，0.5 Mの3種類，浸出温度を25 ℃（常温）とした．浸出開始から15，30，60，90，120分経過した時点で溶液を採取し，ICP-MSで各元素濃度を分析した．

## アンモニア-炭酸アンモニウム溶液による浸出実験

　アンモニア-炭酸アンモニウム溶液による浸出実験では，未焙焼試料および焙焼試料（600 ℃，800 ℃，空気雰囲気，炭素付加率10 wt. %，3時間）を用い，アンモニア-炭酸アンモニウム溶液の濃度を3段階（2 % NH~3~-1.2 % CO~2~，7 % NH~3~-4.2 % CO~2~，15 % NH~3~-9 % CO~2~），溶液温度を25 ℃，50 ℃，75 ℃とした．

## 湿式製錬プロセスの比較評価

　各製錬プロセスの経済性を評価するため，ターゲットメタルの浸出率，試薬コスト，エネルギーコスト，処理時間，および有用金属の回収率に基づく収益性を比較検討した．評価対象としたプロセスは，硫酸浸出法，シュウ酸浸出法，およびアンモニア-炭酸アンモニウム溶液浸出法の3種類である．
