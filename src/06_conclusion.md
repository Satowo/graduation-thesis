# 結論

　本研究では，南鳥島周辺海域産マンガンノジュールの商業利用に向けた製錬プロセスの最適化を目的として，硫酸浸出法，シュウ酸浸出法，およびアンモニア-炭酸アンモニウム溶液浸出法の3種類の手法を比較検討した．

　まず，試料の元素組成を分析した結果，主要成分は Mn（16.11 wt %）および Fe（17.00 wt %）であり，有用金属である Co，Ni，Cu の含有量はそれぞれ 3,712 ppm，2,008 ppm，1,654 ppm であることが明らかになった．

　各浸出法の実験結果から，以下のことが明らかになった．硫酸浸出法では，100 ℃，2 M，120分の条件において，Ni および Cu の回収率が約100 % と非常に高く，回収可能な金属量が多いことが確認された．一方，Co の回収率は約23 % と低く，これが本手法の主な課題となった．シュウ酸浸出法では，0.3 M，25 ℃（常温），120分の条件において，Ni，Co，Cu の全てにおいて高い回収率（約100 %）を示し，特に Co の回収率が高いことが特徴的であった．しかし，シュウ酸は Mn および Fe も同時に溶解させるため，浸出後の処理工程において Mn および Fe を除去する必要がある．この除去過程において，容易にシュウ酸塩が形成され，その中に Ni，Co，Cu が含まれてしまうという重大な問題があることが判明した．アンモニア-炭酸アンモニウム溶液浸出法では，未焙焼試料，50 ℃，15 % NH~3~-9 % CO~2~，120分の条件において最も良い結果が得られたものの，有用金属の回収率は全体的に低く（Cu: 約46 %，Ni: 約7.5 %，Co: 約2.1 %），実用化にはさらなる改善が必要であることが明らかになった．

　経済性の評価において，試薬コスト，エネルギーコスト，処理時間，および回収可能金属量を総合的に比較した結果，硫酸浸出法が最も収益性が高いことが示された．シュウ酸浸出法はエネルギーコストが低いものの，Mn および Fe の除去過程におけるシュウ酸塩形成による有用金属の損失という技術的な課題が解決されない限り，硫酸浸出法に比べて優れているとは言えない．アンモニア-炭酸アンモニウム溶液浸出法は，試薬コストは低いものの，回収率の低さとエネルギーコストの高さにより，現時点での収益性は他の2つのプロセスと比較して低い．

　以上の結果から，南鳥島産マンガンノジュールの商業利用においては，硫酸浸出法（100 ℃，2 M，120分）が最も適していると考えられる．ただし，Co の回収率が低い（約23 %）ことが課題であり，Co の回収率向上のための技術開発が今後の重要な研究課題である．また，シュウ酸浸出法における Mn および Fe の除去過程でのシュウ酸塩形成による有用金属損失の問題を解決する技術の開発も，Co の高回収率を実現する上で重要な課題である．

　本研究により，南鳥島産マンガンノジュールの商業利用に向けた製錬プロセスの指針が提示されたが，実用化に向けては，Co の回収率向上技術の開発，環境対策コストの評価，スケールアップ試験など，さらなる検討が必要である．
