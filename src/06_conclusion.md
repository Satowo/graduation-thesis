# 結論

　本研究では，南鳥島周辺海域産マンガンノジュールに対し，硫酸浸出法，シュウ酸浸出法，アンモニア-炭酸アンモニウム溶液浸出法の3手法を比較し，回収性能と経済性を評価した．試料の主要成分は Mn 16.11 wt %，Fe 17.00 wt %，有用金属は Co 3,712 ppm，Ni 2,008 ppm，Cu 1,654 ppm であった．

　硫酸浸出（100 ℃，2 M，120分）は Ni・Cu をほぼ100 %回収する一方，Co は約23 %にとどまり，Co 回収が主要課題となった．シュウ酸浸出（0.3 M，25 ℃，120分）は Ni・Co・Cu をほぼ100 %回収したが，Mn・Fe も溶出し，後処理でシュウ酸塩が生成して Ni/Co/Cu も共沈してしまう可能性があり、技術課題が残ることが判明した．アンモニア-炭酸アンモニウム浸出（未焙焼，50 ℃，15 % NH~3~-9 % CO~2~，120分）は Cu 約46 %，Ni 約7.5 %，Co 約2.1 %と回収率が低かった．

　経済性の比較では，試薬・エネルギー・処理時間・回収価値を総合すると硫酸浸出が最も損失幅が小さく（回収価値約97 USDに対しコスト約302 USD/t），相対的に有利であった．シュウ酸浸出は常温運転でエネルギー負荷が小さいが，シュウ酸塩による金属損失と高い試薬コストにより大きな赤字（▲1,648 USD/t）となった．アンモニア-炭酸アンモニウム浸出は回収価値が小さく，試薬・加熱・焙焼コストの合計が最も高く，収益性が最も低かった（▲34,993～▲34,999 USD/t）。現状ではいずれも黒字化しておらず，コスト低減と回収率改善が不可欠である．

　以上より，現段階で商業利用に最も近いのは硫酸浸出法（100 ℃，2 M，120分）であるが，Co 回収率の向上が不可欠である．併せて，シュウ酸浸出におけるシュウ酸塩生成による有用金属損失の抑制，アンモニア系の回収率向上と焙焼・加熱コスト低減，環境対策コストの精査，およびスケールアップ試験が今後の重要な検討課題となる．
