# 結論

　本研究では，南鳥島周辺海域産マンガンノジュールに対し，硫酸浸出法，シュウ酸浸出法，アンモニア-炭酸アンモニウム溶液浸出法の3手法を比較し，回収性能と経済性を評価した．試料の主要成分は Mn 16.11 wt %，Fe 17.00 wt %，有用金属は Co 3,712 ppm，Ni 2,008 ppm，Cu 1,654 ppm であった．

　硫酸浸出法では，未焙焼試料を用いた場合，Ni・Cu をほぼ100 %回収する一方，Co は約23 %にとどまり，Co 回収が主要課題となった．しかし，600 ℃，空気雰囲気，炭素付加率10 wt. % で3時間の還元焙焼処理を行うことで，Mn^4+^をMn^2+^に還元し，結晶構造を変化させることで浸出性が大幅に改善された．焙焼試料を用いた硫酸浸出（75 ℃，2 M，180分）では，Ni は94.8 %，Cu は98.3 %，Co は94.8 % の高い浸出率を示し，Co 回収率の課題が解決された．さらに，Mn も61.4 % の浸出率で回収可能となった．シュウ酸浸出法（0.3 M，25 ℃，120分）は Ni・Co・Cu をほぼ100 %回収したが，Mn・Fe も溶出し，後処理でシュウ酸塩が生成して Ni/Co/Cu も共沈してしまう可能性があり，技術課題が残ることが判明した．アンモニア-炭酸アンモニウム溶液浸出法（未焙焼，50 ℃，15 % NH~3~-9 % CO~2~，120分）は Cu 約46 %，Ni 約7.5 %，Co 約2.1 % と回収率が著しく低く，評価対象から除外した．

　経済性の比較では，試薬コスト，エネルギーコスト，処理時間，および有用金属の回収率に基づく収益性を総合的に評価した．硫酸浸出法（焙焼試料，75 ℃，2 M，180分）は，Ni，Co，Cu の回収価値約195 USDに加え，マンガンを硫酸マンガンとして回収することで追加で約247 USD の価値が得られ，合計回収価値は約442 USD となった．これに対し，浸出用試薬157 USD，マンガン回収のための溶媒抽出用試薬約69 USD，加熱・焙焼164～170 USD で総コスト約390～396 USD，差引約46～52 USD/t となり，唯一収益性がプラスとなった．シュウ酸浸出法（0.3 M，25 ℃）は回収価値約222 USD に対し，試薬1,870 USD（加熱0 USD）で差引▲1,648 USD/t となった．

　以上より，南鳥島のマンガンノジュールに対しては，焙焼→硫酸浸出→溶媒抽出による分離という処理フローが最も適していると結論づけられる．具体的には，まず還元焙焼処理により浸出性を向上させ，次に硫酸浸出により Ni，Co，Cu を高回収率で回収し，さらに Seol et al. (2024) が報告した方法に従い，Cyanex 272とD2EHPAを用いた溶媒抽出により Co(II)，Mn(II)，Ni(II)を分離することで，純度99.9 %以上のMnSO~4~溶液として回収できる．この処理フローにより，現段階で商業利用に向けた有望な選択肢となった．今後の検討課題としては，浸出しなかったMnの回収による収益性のさらなる改善，スケールアップ試験，環境対策コストの精査，および実用化に向けたプロセス最適化が挙げられる．
