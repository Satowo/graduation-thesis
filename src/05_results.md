# 実験結果及び考察

## 酸分解

　はじめに，試料の元素組成を特定するため，酸分解処理後にICP-MS（^57^Fe等を測定）を用いた分析を行った．分析結果（4回の平均値）をTable. 5-1に示す．

　本研究で用いたマンガンノジュール試料の主要成分は Mn および Fe であり，含有量はそれぞれ 161,100 ppm (16.11 wt %) および 170,000 ppm (17.00 wt %) であった．また，有用金属である Co，Ni，Cu の含有量は，それぞれ 3,712 ppm，2,008 ppm，1,654 ppm であった．これら 5 元素の合計含有量は約 34 wt % であり，残りの大部分は不溶残渣や水分などの非金属成分であると考えられる．

## 硫酸浸出実験

　硫酸を用いた浸出実験の結果をFig. 5-1からFig. 5-5に示す．

　はじめに，有用金属である Ni，Co，Cu の浸出挙動について述べる．Ni (Fig. 5-1) は，100 ℃ においては浸出がより促進され，全ての濃度条件で最終的に 100 % 近い浸出率に達した．特に 1 M および 2 M では浸出開始 120 分以内にほぼ完全に溶解した．Cu (Fig. 5-3) は，全ての実験条件において浸出開始直後から 100 % に近い浸出率を示した．これは，本試料中の Cu 成分が低濃度の硫酸に対しても非常に溶解しやすい形態で存在していることを示唆している．これに対し，Co (Fig. 5-2) の浸出率は100 ℃，2 M の条件で最大 23.3 % (120 分時点) まで上昇したものの，全体としてはNi，Coに比べると低い浸出率であった．

　次に，主要成分である Mn および Fe の浸出挙動について述べる．Fe (Fig. 5-4) の浸出率は温度および硫酸濃度に強く依存した．25 ℃ における 180 分後の浸出率は，0.5 M で 22.5 %，1 M で 36.4 %，2 M で 52.1 % であったが，100 ℃ では，0.5 M で 31.0 %，1 M で 61.9 %，2 M で 81.5 % まで上昇した．一方，Mn (Fig. 5-5) はいずれの条件でも 1.1 ～ 1.7 % 程度に留まった．

## シュウ酸による浸出実験

　シュウ酸を用いた浸出実験の結果をFig. 5-6からFig. 5-10に示す．

　はじめに，有用金属である Ni，Co，Cu の浸出挙動について述べる．Ni (Fig. 5-6) は，0.3 M および 0.5 M において 100 % を超える高い浸出率を示した．0.1 M では 15 分時点で 99.4 %，その後わずかに低下し，120 分時点で 95.5 % であった．0.3 M および 0.5 M では，15 分時点でそれぞれ 110.6 % および 112.2 % と高い値を示し，その後わずかに低下したが，120 分時点でも 102.0 % および 104.9 % と高い浸出率を維持した．Cu (Fig. 5-8) は，0.1 M では 15 分時点で 100.9 %，120 分時点で 95.7 % であった．0.3 M および 0.5 M では，15 分時点でそれぞれ 106.9 % および 107.6 % と高い値を示し，120 分時点でも 99.6 % および 101.7 % と高い浸出率を維持した．Co (Fig. 5-7) も同様の傾向を示し，0.1 M では 15 分時点で 100.5 %，120 分時点で 95.4 % であった．0.3 M および 0.5 M では，15 分時点でそれぞれ 110.2 % および 112.0 % と高い値を示し，120 分時点でも 103.7 % および 104.0 % と高い浸出率を維持した．なお，これらの 100 % を超える浸出率は，ICP-MS の測定誤差や試料の偏りによるものと考えられる．

　次に，主要成分である Mn および Fe の浸出挙動について述べる．Mn (Fig. 5-9) は，0.3 M および 0.5 M において 100 % を超える高い浸出率を示した．0.1 M では 15 分時点で 99.1 %，その後わずかに低下し，120 分時点で 94.3 % であった．0.3 M および 0.5 M では，15 分時点でそれぞれ 113.8 % および 116.1 % と高い値を示し，その後わずかに低下したが，120 分時点でも 106.5 % および 106.4 % と高い浸出率を維持した．一方，Fe (Fig. 5-10) の浸出挙動は他の元素と大きく異なった．0.1 M では 15 分時点で 67.5 %，その後わずかに低下し，120 分時点で 63.2 % と比較的低い浸出率に留まった．これに対し，0.3 M および 0.5 M では，15 分時点でそれぞれ 88.3 % および 91.1 % と高い値を示し，その後わずかに上昇し，120 分時点でそれぞれ 91.4 % および 93.6 % と高い浸出率に達した．

## 焙焼処理およびXRDによる構造分析

## アンモニア-炭酸アンモニウム溶液による浸出実験

　アンモニア-炭酸アンモニウム溶液を用いた浸出実験の結果をFig. 5-11からFig. 5-25に示す．未焙焼試料および600 ℃，空気雰囲気，炭素付加率10 wt. % で3時間焙焼した試料（No214）の2種類について実験を行った．この実験により，溶液濃度が高い方が浸出率は高いと考え，濃度を15 % NH~3~-9 % CO~2~ の溶液に固定し，試料を800 ℃，空気雰囲気，炭素付加率10 wt. % で3時間焙焼したもの（No148）を用いて実験を追加で行った．

　はじめに，有用金属である Ni，Co，Cu の浸出挙動について述べる．未焙焼試料における Cu (Fig. 5-13) は，全ての条件で最も高い浸出率を示した．25 ℃ では，2 % NH~3~-1.2 % CO~2~ 条件で 120 分時点で 8.15 %，7 % NH~3~-4.2 % CO~2~ 条件で 26.69 %，15 % NH~3~-9 % CO~2~ 条件で 28.69 % であった．50 ℃ では，2 % NH~3~-1.2 % CO~2~ 条件で 10.90 %，7 % NH~3~-4.2 % CO~2~ 条件で 37.39 %，15 % NH~3~-9 % CO~2~ 条件で 46.21 % と，温度上昇により浸出率が向上した．一方，焙焼試料（No214）における Cu (Fig. 5-18) は，未焙焼試料と比較して大幅に低下し，全ての条件で 2.22 % 以下に留まった．最高値は 50 ℃，15 % NH~3~-9 % CO~2~ 条件の 60 分時点で 2.22 % であったが，120 分時点では 1.42 % に低下した．さらに，800 ℃ で焙焼した試料（No148）における Cu (Fig. 5-23) は，25 ℃ および 50 ℃ では低い浸出率を示し，120 分時点でそれぞれ 0.03 % および 0.03 % であった．一方，80 ℃ では 60 分時点で 1.01 % と高いピークを示したが，その後急激に低下し，120 分時点では -0.02 % と負の値を示した．

　未焙焼試料における Ni (Fig. 5-11) の浸出率は，25 ℃ では 2 % NH~3~-1.2 % CO~2~ 条件で 120 分時点で 1.62 %，7 % NH~3~-4.2 % CO~2~ 条件で 3.39 %，15 % NH~3~-9 % CO~2~ 条件で 2.87 % であった．50 ℃ では，2 % NH~3~-1.2 % CO~2~ 条件で 1.64 %，7 % NH~3~-4.2 % CO~2~ 条件で 6.95 %，15 % NH~3~-9 % CO~2~ 条件で 7.53 % と，温度上昇により浸出率が向上した．焙焼試料（No214）における Ni (Fig. 5-16) は，全ての条件で負の値を示し，浸出がほとんど認められなかった．800 ℃ で焙焼した試料（No148）における Ni (Fig. 5-21) も，25 ℃ および 50 ℃ では負の値を示し，120 分時点でそれぞれ -0.09 % および -0.04 % であった．一方，80 ℃ では 60 分時点で 0.44 % と高いピークを示したが，その後急激に低下し，120 分時点では -0.03 % と負の値を示した．

　未焙焼試料における Co (Fig. 5-12) の浸出率は，全体的に低く，25 ℃ では 2 % NH~3~-1.2 % CO~2~ 条件で 120 分時点で 0.10 %，7 % NH~3~-4.2 % CO~2~ 条件で 0.29 %，15 % NH~3~-9 % CO~2~ 条件で 0.84 % であった．50 ℃ では，2 % NH~3~-1.2 % CO~2~ 条件で 0.15 %，7 % NH~3~-4.2 % CO~2~ 条件で 0.83 %，15 % NH~3~-9 % CO~2~ 条件で 2.09 % と，温度上昇により浸出率が向上した．焙焼試料（No214）における Co (Fig. 5-17) は，未焙焼試料と同様に低い浸出率を示し，25 ℃ では 0.22 ～ 0.39 %，50 ℃ では 0.31 ～ 0.54 % の範囲に留まった．800 ℃ で焙焼した試料（No148）における Co (Fig. 5-22) は，25 ℃ および 50 ℃ では低い浸出率を示し，120 分時点でそれぞれ 0.02 % および 0.02 % であった．一方，80 ℃ では 60 分時点で 0.21 % と高いピークを示したが，その後急激に低下し，120 分時点では 0.05 % に低下した．

　次に，主要成分である Mn および Fe の浸出挙動について述べる．未焙焼試料における Mn (Fig. 5-14) および Fe (Fig. 5-15) は，全ての条件で浸出率が 0.08 % 以下と非常に低く，実質的に浸出が認められなかった．焙焼試料（No214）における Mn (Fig. 5-19) および Fe (Fig. 5-20) も同様に低い浸出率を示し，Mn は 0.08 % 以下，Fe は 0.11 % 以下に留まった．800 ℃ で焙焼した試料（No148）における Mn (Fig. 5-24) は，25 ℃ および 50 ℃ では低い浸出率を示し，120 分時点でそれぞれ 0.01 % および 0.02 % であった．80 ℃ では 60 分時点で 0.04 % とピークを示したが，その後低下し，120 分時点では 0.02 % に低下した．Fe (Fig. 5-25) は，25 ℃ および 50 ℃ では非常に低い浸出率を示し，120 分時点でそれぞれ 0.001 % および 0.006 % であった．80 ℃ では 60 分時点で 0.034 % とピークを示したが，その後低下し，120 分時点では 0.008 % に低下した．

## コスト比較

　各製錬プロセスの経済性を評価するため，試薬コスト，エネルギーコスト，処理時間，および有用金属の回収率に基づく収益性を比較検討した．評価対象としたプロセスは，硫酸浸出法，シュウ酸浸出法，およびアンモニア-炭酸アンモニウム溶液浸出法の3種類である．

### 各プロセスの最適条件

　各プロセスの最適条件は以下の通りである．硫酸浸出法では，100 ℃，2 M，120分の条件で最も良い結果が得られた．シュウ酸浸出法では，0.3 M，25 ℃（常温），120分の条件で最も良い結果が得られた．アンモニア-炭酸アンモニウム溶液浸出法では，未焙焼試料，50 ℃，15 % NH~3~-9 % CO~2~，120分の条件で最も良い結果が得られた．

### 回収可能金属量の比較

　試料1トンあたりの各金属含有量（Ni: 2,008 ppm，Co: 3,712 ppm，Cu: 1,654 ppm）に各プロセスの最適条件における浸出率を乗じることで，回収可能な金属量を算出した．硫酸浸出法では，Ni は約 2.0 kg，Cu は約 1.7 kg，Co は約 0.9 kg が回収可能であった．シュウ酸浸出法では，Ni は約 2.0 kg，Cu は約 1.7 kg，Co は約 3.9 kg が回収可能であった．アンモニア-炭酸アンモニウム溶液浸出法では，Cu は約 0.8 kg，Ni は約 0.2 kg，Co は約 0.08 kg が回収可能であった．

### 試薬コストの比較

　処理1トンのマンガンノジュール試料に必要な試薬コストを比較した結果，硫酸浸出法では，硫酸の使用量が最も多く，試薬コストが最も高くなった．シュウ酸浸出法では，シュウ酸の使用量は中程度であったが，試薬単価が硫酸と比較して高いため，試薬コストは硫酸浸出法と同程度またはやや高くなった．アンモニア-炭酸アンモニウム溶液浸出法では，アンモニアおよび二酸化炭素の使用量は比較的少なく，試薬コストは3つのプロセスの中で最も低かった．ただし，還元焙焼を必要とする場合，還元剤として使用する炭素（黒鉛）のコストが追加される．

### エネルギーコストの比較

　各プロセスのエネルギーコストを比較した結果，硫酸浸出法では，100 ℃での加熱に必要なエネルギーコストが発生したが，処理時間が短い（120分）ため，総エネルギーコストは中程度であった．シュウ酸浸出法では，25 ℃での常温処理のため，加熱エネルギーは不要であり，エネルギーコストは3つのプロセスの中で最も低かった．アンモニア-炭酸アンモニウム溶液浸出法では，還元焙焼工程（600 ℃または800 ℃，3時間）に必要なエネルギーコストが大きく，さらに浸出時の加熱（50 ℃）に必要なエネルギーコストも加わるため，総エネルギーコストは3つのプロセスの中で最も高くなった．

### 処理時間の比較

　各プロセスの処理時間を比較した結果，硫酸浸出法では，120分で高い浸出率が得られ，処理時間は最も短かった．シュウ酸浸出法では，120分で高い浸出率が得られ，処理時間は硫酸浸出法と同程度であった．アンモニア-炭酸アンモニウム溶液浸出法では，還元焙焼工程（3時間）に加えて，浸出時間（120分）が必要であり，総処理時間は約5時間と最も長くなった．

### 収益性の総合評価

　各プロセスの収益性を総合的に評価した結果，以下のことが明らかになった．硫酸浸出法は，Ni および Cu の回収率が非常に高く（約100 %），回収可能な金属量が多いため，収益性が高いと考えられた．一方，Co の回収率が低い（約23 %）ことが課題となった．シュウ酸浸出法は，Ni，Co，Cu の全てにおいて高い回収率（約100 %）を示し，特に Co の回収率が高いことが特徴的であった．しかし，シュウ酸は Mn および Fe も同時に溶解させるため，浸出後の処理工程において Mn および Fe を除去する必要がある．この除去過程において，容易にシュウ酸塩が形成され，その中に Ni，Co，Cu が含まれてしまうという重大な問題がある．この問題が解決されない限り，シュウ酸浸出法は硫酸浸出法に比べて優れているとは言えないと考えられた．アンモニア-炭酸アンモニウム溶液浸出法は，試薬コストは低いものの，有用金属の回収率が全体的に低く，さらにエネルギーコストが高いため，現時点での収益性は他の2つのプロセスと比較して低いと考えられた．

### プロセス選択の指針

　以上の結果から，南鳥島産マンガンノジュールの商業利用においては，硫酸浸出法（100 ℃，2 M，120分）が最も適していると考えられる．ただし，Co の回収率が低い（約23 %）ことが課題である．
