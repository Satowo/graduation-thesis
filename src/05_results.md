# 実験結果及び考察

## 酸分解

　はじめに，試料の元素組成を特定するため，酸分解処理後にICP-MS（^57^Fe等を測定）を用いた分析を行った．分析結果（4回の平均値）を表5-1に示す．

　本研究で用いたマンガンノジュール試料の主要成分はMnおよびFeであり，含有量はそれぞれ161,100 ppm（16.11 wt%）および170,000 ppm（17.00 wt%）であった．また，有用金属であるCo，Ni，Cuの含有量は，それぞれ3,712 ppm，2,008 ppm，1,654 ppmであった．これら5元素の合計含有量は約34 wt%であり，残りの大部分は不溶残渣や水分などの非金属成分であると考えられる．

表5-1　試料の元素組成
| 元素 | 含有量 [ppm] | 含有量 [wt%] |
| :---: | :---: | :---: |
| Mn | 161,100 | 16.11 |
| Fe | 170,000 | 17.00 |
| Co | 3,712 | 0.3712 |
| Ni | 2,008 | 0.2008 |
| Cu | 1,654 | 0.1654 |
