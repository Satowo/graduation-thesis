# 研究手法

## 試料
　実験試料は，2024 年に日本財団と東京大学の共同調査 (日本財団，2024) によって南鳥島海域で採取されたマンガンノジュール試料である．本航海で得られたマンガンノジュール試料15 個をジョークラッシャー (製品名) で粉砕し，試験用ふるいで粒径を150 μm 以下に揃えたものを化学分析および湿式処理試験に使用した．

## 検量線用標準溶液の調製

　検量線用標準溶液の調製には，Machida et al. (2016) が南鳥島周辺海域の日本排他的経済水域内で採取したマンガンノジュールの元素組成データ (Table 3) を用いた．同論文では，Site 6K 1207 で採取されたマンガンノジュール試料 (R01，R02，R06) について，主要元素および微量元素の組成が報告されている．これらの組成値に基づいて，各元素の標準溶液を調製した．

## 試料の酸分解

　はじめに，試料の元素組成を調べるため，以下の手順で酸分解を行った．

### 酸分解手順

　110 ℃のエアバス内で12 時間以上乾燥させた4つの蓋付きテフロンジャーに粉末試料を約0.050 g を計量し，過塩素酸 (HClO~4~) 800 μL を加え，すぐに蓋をして，テフロンジャーを回しながら壁面についた試料粉末を全て液中に回収した．次にフッ酸4.0 mL，硝酸2.0 mL を加えた．蓋を閉めてホットプレートに載せ，130 ℃で2 時間以上加熱した．一旦ホットプレートから下ろして冷ましてから，蓋を開けて110 ℃で12 時間以上加熱した．次に，ホットプレートの設定を160 ℃に上げ，3 時間以上加熱した．さらに190 ℃まで上げ，ゼリー状になるまで乾固させた．ホットプレートから降ろして冷ました後，HCl:HNO~3~=1.5 mL:0.5 mL を加えて王水処理を行った．蓋をしてホットプレートに載せ，90 ℃で6 時間以上加熱した．テフロンジャーの蓋を開け，120 ℃で2 時間程度加熱した後，160 ℃に上げてゼリー状になるまで乾固させた．ホットプレートから降ろして冷ました後，回収用酸 (2 % 木村溶液) 10 mL を加えた．再び蓋をしてテフロンジャーをホットプレートに載せ，90 ℃で3 時間以上加熱した．完全に溶けていることを確認した後，蓋をしたままで重さを量り，これを Stock solution とした．

### 希釈

　プラボトルに希釈用混酸9.8 mL，Stock solution 0.2 mL を入れた．最終希釈倍率がおよそ10 万倍となるようこの1 次希釈溶液をさらに10 倍希釈し，その溶液を ICP-MS を用いて分析した．4つの分析結果の平均値を後述の実験での浸出率計算に用いた．

## 焙焼処理およびXRDによる構造分析

　マンガンノジュールの還元焙焼に伴う結晶構造の変化を評価するため，未焙焼試料および種々の条件下で焙焼処理を施した試料に対して X 線回折 (XRD) 分析を行った．
　焙焼処理では，粉末試料に還元剤として黒鉛を添加し，マッフル炉を用いて加熱を行った．実験条件として，焙焼温度は500 ℃，600 ℃，700 ℃の3 条件，雰囲気は大気 (空気) および窒素の2 条件を設定した．活性炭の添加量 (炭素付加率) は，試料重量に対して3，5，10 wt. % の3 段階とした．加熱プログラムは，昇温速度10 K/min で目的温度まで昇温した後，30 分間等温保持する条件とした．
　さらに，焙焼時間の影響および高温域での相変化を詳細に検討するため，炭素付加率10 wt. %，大気雰囲気において，焙焼温度600 ℃で保持時間を30，60，120，180 分とした条件，および焙焼温度800 ℃で保持時間を30 分とした条件についても追加で実験を行った．
　XRD 分析は，未焙焼試料および各条件で得られた焙焼試料を対象として実施し，回折パターンの比較により，構成鉱物の同定および焙焼に伴う相転移の過程を検討した．

## 硫酸による浸出実験

　硫酸による浸出実験では，硫酸の濃度を0.5 M，1 M，2 M の3 種類，攪拌温度を25 ℃および100 ℃の2 条件とした．所定温度に加熱したスターラーの上に，攪拌子を入れたテフロンジャー3 つを配置した．各テフロンジャーに試料を1 g，硫酸を10 mL 加え，この時点を浸出開始時刻とした．浸出開始から15 分，30 分，60 分，120 分，180 分経過した時点で，シリンジを用いて溶液を採取し，シリンジフィルターによって濾過して濾液を回収した．全ての濾液を総希釈倍率50,000 倍となるよう希釈を行い，ICP-MS で各元素濃度を分析した．その分析結果と酸分解の分析結果から Cu，Ni，Co，Mn，Fe の浸出率を計算した．

## シュウ酸による浸出実験

　シュウ酸による浸出実験では，未焙焼のマンガンノジュール試料を用い，シュウ酸濃度を0.1 M，0.3 M，0.5 M の3 種類，浸出温度を25 ℃とした．スターラーの上に攪拌子を入れたビーカーを配置し，各ビーカーにシュウ酸を49.5 g 入れ，試料を0.5 g 加えた時点を浸出開始時刻とした．浸出開始から15，30，60，90，120 分経過した時点で，pH および ORP を測定・記録した後，シリンジを用いて溶液を約1 mL 採取し，シリンジフィルターで濾過して濾液を回収した．得られた全濾液を総希釈倍率50,000 倍となるよう希釈し，ICP-MS で各元素濃度を分析した．その分析結果と酸分解の分析結果から Cu，Ni，Co，Mn，Fe の浸出率を計算した．

## アンモニア-炭酸アンモニウム溶液による浸出実験

　アンモニア-炭酸アンモニウム溶液による浸出実験では，未焙焼のマンガンノジュール試料および10 wt. % の炭素を加えて600 ℃で3 時間焙焼した試料を用い，アンモニア-炭酸アンモニウム溶液の濃度を3 段階 (2 % NH~3~-1.2 % CO~2~，7 % NH~3~-4.2 % CO~2~，15 % NH~3~-9 % CO~2~)，溶液温度を25 ℃および50 ℃の2 条件，パルプ濃度を1 % (河原 et al.，1985) とした．スターラーの上に攪拌子を入れたビーカーを配置し，各濃度の溶液49.5 mL に対し，試料を0.5 g 加えた時点を浸出開始時刻とした．浸出開始から15，30，60，90，120 分経過した時点で，pH および ORP を測定・記録した後，シリンジを用いて溶液を約1 mL 採取し，シリンジフィルターで濾過して濾液を回収した．得られた全濾液を総希釈倍率50,000 倍となるよう希釈し，ICP-MS を用いて分析した．その分析結果と酸分解の分析結果から Cu，Ni，Co，Mn，Fe の浸出率を計算した．

## コスト比較

　各製錬プロセスの経済性を評価するため，試薬コスト，エネルギーコスト，処理時間，および有用金属の回収率に基づく収益性を比較検討した．評価対象としたプロセスは，硫酸浸出法，シュウ酸浸出法，およびアンモニア-炭酸アンモニウム溶液浸出法の3種類である．

### 試薬コストの評価

　各プロセスで使用する試薬の単価を調査し，処理1トンのマンガンノジュール試料に必要な試薬量から試薬コストを算出した．硫酸浸出法では，硫酸 (H~2~SO~4~) の使用量を浸出条件（濃度0.5 M，1 M，2 M）に応じて計算した．シュウ酸浸出法では，シュウ酸 (H~2~C~2~O~4~) の使用量を浸出条件（濃度0.1 M，0.3 M，0.5 M）に応じて計算した．アンモニア-炭酸アンモニウム溶液浸出法では，アンモニア (NH~3~) および二酸化炭素 (CO~2~) の使用量を溶液濃度（2 % NH~3~-1.2 % CO~2~，7 % NH~3~-4.2 % CO~2~，15 % NH~3~-9 % CO~2~）に応じて計算した．また，還元焙焼を必要とするアンモニア浸出法では，還元剤として使用する炭素（黒鉛）のコストも考慮した．

### エネルギーコストの評価

　各プロセスで消費されるエネルギーを評価するため，以下の項目を考慮した．硫酸浸出法では，100 ℃での加熱に必要なエネルギーコストを計算した．シュウ酸浸出法では，25 ℃での常温処理のため，加熱エネルギーは不要であると仮定した．アンモニア-炭酸アンモニウム溶液浸出法では，還元焙焼工程（600 ℃または800 ℃，3時間）に必要なエネルギーコスト，および浸出時の加熱（25 ℃または50 ℃）に必要なエネルギーコストをそれぞれ計算した．エネルギーコストの算出には，電気料金単価および燃料費単価を用いた．

### 処理時間の評価

　各プロセスの処理時間を比較するため，浸出実験で得られた最適条件における浸出時間を基準とした．硫酸浸出法では，100 ℃，2 M 条件における120分時点での浸出率を基準とした．シュウ酸浸出法では，0.3 M 条件における120分時点での浸出率を基準とした．アンモニア-炭酸アンモニウム溶液浸出法では，15 % NH~3~-9 % CO~2~ 条件における120分時点での浸出率を基準とした．また，還元焙焼を必要とするプロセスでは，焙焼時間（3時間）も処理時間に含めた．

### 回収率に基づく収益性の評価

　各プロセスで回収可能な有用金属（Ni，Co，Cu）の量を，実験結果から得られた浸出率に基づいて算出した．試料1トンあたりの各金属含有量（Ni: 2,008 ppm，Co: 3,712 ppm，Cu: 1,654 ppm）に各プロセスの浸出率を乗じることで，回収可能な金属量を計算した．次に，各金属の市場価格（円/kg）を調査し，回収可能な金属量に市場価格を乗じることで，試料1トンあたりの収益を算出した．最終的に，試薬コストおよびエネルギーコストの合計を収益から差し引くことで，各プロセスの収益性を評価した．

### コスト比較の前提条件

　コスト比較を行うにあたり，以下の前提条件を設定した．試薬の単価は，工業用試薬の市場価格（2024年時点）を基準とした．エネルギーコストは，日本の平均的な電気料金および燃料費単価を基準とした．金属の市場価格は，2024年の平均価格を基準とした．設備コストおよび人件費については，本評価の対象外とした．また，環境対策コスト（廃液処理，排ガス処理など）についても，本評価の対象外とした．各製錬プロセスの経済性を評価するため，試薬コスト，エネルギーコスト，処理時間，および有用金属の回収率に基づく収益性を比較検討した．評価対象としたプロセスは，硫酸浸出法，シュウ酸浸出法，およびアンモニア-炭酸アンモニウム溶液浸出法の3種類である．
